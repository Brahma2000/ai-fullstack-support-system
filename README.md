# AI Full Stack Support System

This project demonstrates how to incrementally add AI capabilities to a traditional full-stack application using Spring Boot and React.

## Features

- Create support tickets
- AI-based ticket analysis
- Automatic priority detection
- AI usage monitoring
- REST API documentation with Swagger
- H2 in-memory database

## Tech Stack
- Frontend: React.js
- Backend: Spring Boot
- AI: OpenAI / Azure OpenAI
- Database: PostgreSQL

  ## Roadmap
  - Week 1: Achitecture & Planning
  - Week 2: Backend Ticket CRUD
  - Week 3: AI Backend Integration
  - Week 4: Prompt Engineering & Structured Output
  - Week 5: Monitoring and usage limits
  - Week 6: Swagger API Documentation
