# Week 6 - API Documentation with Swagger

## Objective
Integrate Swagger (OpenAPI) for automatic REST API documentation and testing.

## Tools Used
- Spring Boot
- SpringDoc OpenAPI
- Swagger UI

## Implementation Steps
1. Added Swagger dependency in 'pom.xml'
2. Created OpenApiConfig class
3. Added API documentation annotations in TicketController
4. Verified endpoints in Swagger UI

## Swagger URL
http://localhost:8080/swagger-ui/index.html

## Available Endpoints
POST /tickets
GET /tickets
GET /tickets/{id}
