# Week 7 - Frontend Development (React)

## Objective
Build a React frontend to interact with the backend API.

## Features Implemented

- Ticket creation form
- Fetch tickets from backend
- Display ticket list
- Show AI-generated summary and priority

## Tech Stack

- React
- Axios
- REST API

## API Integration

Frontend connects to:

http://localhost:8080/tickets

## Components

- TicketForm.js -> Create ticket
- TicketList.js -> Display tickets
- api.js -> API calls

## Testing

1. Start backend
2. Start frontend
3. Create ticket
4. Verify ticket list updates

## Outcome

The project is now a full-stack application with:
- Spring Boot backend
- React frontend
- AI-powered ticket system
